# Demo flow

A self-contained demo against the bundled login page in `examples/login-demo/`.
Run it with `./run.sh` (which serves the page and the model for you).

## Sign in page renders and reacts
- goto /
- verify a Sign in heading is visible
- verify an email input field is visible
- verify not an error message is shown
- click the blue "Sign in" button
- verify a welcome message is visible
