# Weak flight recommendation feedback

## Recommend a solution after flight feedback
- goto http://localhost:5173/
- wait for the weak flights to finish loading
- verify the Weak Flights heading is visible
- verify at least one weak flight is listed
- click the first weak flight in the list
- wait for the weak flight details to load
- verify the selected flight's details are visible
- verify at least one recommended solution is visible
- click the top recommended solution
- verify the recommended solution details and feedback controls are visible
- click the positive feedback option for the recommended solution
- type "Recommended solution is relevant for this weak flight." into the feedback comment field
- click the button that submits the feedback
- wait for the feedback success confirmation to appear
- verify the feedback was submitted successfully
- click the "Adaptive Recommendation" link in the sidebar
- wait for the recommendation page to load
- verify the Adaptive Recommendation page is visible
- verify the solution that received positive feedback appears in the recommended solutions list
